    C++

    C++ is a powerful, high-performance programming language that is widely used in software development. It was developed by Bjarne Stroustrup at Bell Labs in the early 1980s as an extension of the C programming language. C++ code is compiled into machine code using a compiler, which makes the programs fast and efficient.

    Python

    Python is a high-level, interpreted programming language known for its readability, simplicity, and versatility. It was created by Guido van Rossum and first released in 1991.Python code is executed line by line by an interpreter.Variables can change type and don't need explicit declaration.Abstracts low-level details, making development faster and easier

    HTML 5

    HTML5 is the latest version of Hypertext Markup Language (HTML), which is the standard language for creating and structuring content on the web. HTML5 introduces new elements, attributes, and behaviors that provide more powerful and flexible ways to develop web pages and applications.HTML5 aims to improve accessibility across device.

    CSS

    CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, colors, fonts, and overall appearance of web pages, making them more visually appealing and easier to interact with.CSS provides tools to create responsive web designs to different screen sizes.
